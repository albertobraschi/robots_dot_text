Module RobotsDotText::UserAgents
In: lib/robots_dot_text/user_agents.rb

When adding User-agent directives with the add method, you can pass any of these symbols as the user agent parameter. For example:

robots_dot_text new { |rules| rules.add :google, "/users", "/admin" }

… which makes life a little easier.
